    File Download

  • Path: /api/files/{path}
  • HTTP Method: GET
  • Example: /api/files/images/holidays/img01.jpg

    File Upload

  • Path: /api/files/{path}
  • HTTP Method: PUT
  • Example: /api/files/images/holidays/img01.jpg

    List Files

  • Path: /api/files/{path}
  • HTTP Method: GET
  • Example: /api/files/images/holidays

    List Folders

  • Path: /api/folders/{path}
  • HTTP Method: GET
  • Example: /api/folders/images/holidays

    Create Folders

  • Path: /api/folders/{path}
  • HTTP Method: GET
  • Example: /api/folders/images/holidays
